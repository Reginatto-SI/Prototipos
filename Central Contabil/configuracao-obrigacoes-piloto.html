Nova obrigação

Cadastro de parâmetros para o sistema gerar automaticamente obrigações recorrentes para as empresas aplicáveis.
Modo criação
Define quais perfis de empresa receberão automaticamente essa obrigação quando o sistema executar a geração recorrente.
Exemplo de regra fiscal: competência janeiro → 2º mês subsequente, dia 15. Essa definição será usada pelo motor para gerar os vencimentos automaticamente.
Exemplo de cálculo visual: competência 01/2026 → vencimento 15/03/2026.
Exemplo completo com fim de semana: competência 01/2026 → prazo fiscal 17/03/2026; prazo interno (2 dias antes) → 15/03/2026. Se o prazo interno cair no domingo: Antecipar → sexta-feira anterior, Postergar → segunda-feira seguinte, Manter data original → mantém domingo como data operacional.
